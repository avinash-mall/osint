# Why a FAIR1M-2.0 OBB specialist

**Decision date:** 2026-05-28
**Status:** SUPERSEDED 2026-05-31 — the FAIR1M-OBB detector was removed.

This layer shipped as a no-op stub (no checkpoint was ever baked, so it produced zero
detections) and its fine-grained niche overlapped DOTA-OBB. It was removed permanently. The
FAIR1M *reference dataset* is unaffected and still feeds the platform-identification corpus.

See **[removed-fair1m-and-remoteclip.md](removed-fair1m-and-remoteclip.md)** for the rationale
and the full list of code/config/doc changes.
